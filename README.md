# AI---Powered-Healthcare-Dashboard

# Project Overview
This Power BI Project analyzes Healthcare operations and patient data using built-in AI capabilities to deliver clear, actionable insights for hospital decision-making.

# KPI Cards
1) Emergency Admission Count.
2) Average Billing per Patient.
3) Total Billing.
4) Total Patients.

# Key Insights
1) Emergency cases from a major share of patient admissions.
2) Patient load is concentrated in a few medical conditions
3) Insured patients drive most of the billing revenue.
4) Billing remains steady across months, showing consistent demand.
5) Filter applied on Medical condition slicer for focused analysis.
6) AI assisted analysis helps surface these patterns quickly.

# AI Capabilities Applied
1) AI assisted insights and intelligent visuals.
2) Dynamic filtering with smart analysis.
3) Automated pattern identification within data.
